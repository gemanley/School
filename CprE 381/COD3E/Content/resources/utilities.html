Utility Software Installation

Adobe Reader

To make use this CD, you will need a PDF viewer such as Adobe Reader or Adobe Acrobat installed. This CD contains recent versions of Adobe Reader for Windows and Macintosh platforms. Visit the Reader homepage for more information.

  • Click here to install Adobe Reader 6.0.1 for Windows XP/ME/2000/NT/98SE.
    Note: you may need administrative privileges to run the installer.
  • Click here to install Adobe Reader 6.0.1 for Mac OS 10.2.2 or higher
  • Adobe Systems Incorporated encourages you to update to the latest release of Adobe Reader, version 6.0.2. Once you have successfully installed Adobe Reader 6.0.1, you may install the new 6.0.2 update. From within Adobe Reader 6.0.1, go to the Help menu, choose Updates, and follow the instructions.
  • Click here to install Adobe Reader 5.1 for Mac OS 8.6 or higher

Note:

  • If you are running Windows 95 or 98 (not Second Edition), and do not have a copy of Adobe Reader, you will need to download version 5.0.5 from the Adobe website.
    Click here to visit the Adobe Reader download archive.
  • You do not need to install Adobe Reader if you already have Adobe Acrobat installed.
  • If supported by your browser, it is strongly recommended that you configure viewing PDF files within the browser using the Reader plugin, rather than launching a separate Reader window. Refer to your browser help for details on plugin configuration.

Browser Plugins

Some of the material included on this CD makes use of Flash® animations. To view this material properly, you will need to have Macromedia® Flash Player installed. If you don't already, you can install the Shockwave® Player (which includes Flash) from this CD.

  • Click here to install Shockwave Player for Windows.
  • Click here to install Shockwave Player for Mac OS X.
  • Click here to install Shockwave Player for Mac OS 9.2.

Note that recent versions of some browsers, including Internet Explorer, Netscape, and AOL include Flash Player "out of the box".

Some of the material included on this CD contains photographic panoramas that can only be viewed with the iseemedia Zoom Viewer browser plugin.

  • Click here to install the Zoom Viewer plugin for Windows.
  • Click here to install the Zoom Viewer plugin for Mac OS.
