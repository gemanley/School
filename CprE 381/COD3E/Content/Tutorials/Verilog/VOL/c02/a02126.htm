Which of the following statements are legal?

1. $time;
2. $display("hello sparky");
3. x = $time;
4. y = $display("hello sparky");
5. if (done) $finish;
6. always @(x) $display($time, " x -> ", x);

Got it. Statement 1 is illegal because $time is a function, not a task. Therefore, it can't stand alone as a statement. Statement 4 is illegal because $display is a task and can't be used as a right-hand side of an expression.
