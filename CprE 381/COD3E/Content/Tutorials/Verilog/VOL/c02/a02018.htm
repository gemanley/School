Which of the following are legal comments?

a. x = y; // this is a comment
b. x = y; /* this is a comment
c. // x = y; /* this line is commented out
d. /* x = y; // this !@#$*/$# is commented out!
e. // x = y; // this !@#$*/$# is commented out!
f. /* x = y; /* this is a comment */ removed!! */

The three highlighted statements are legal. Statement (b) has an unclosed comment, which would cause the following line(s) to be treated as comment until a closing "*/" was encountered. The comments in statements (d) and (f) are prematurely closed by the first "*/" in the line. Subsequent text would be treated as a statement.
