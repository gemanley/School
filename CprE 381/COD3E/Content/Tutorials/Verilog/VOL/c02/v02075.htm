A scalar quantity is a single bit. The value of this bit can be one of

  • 0 - logic zero (false)
  • 1 - logic one (true)
  • x - unknown value
  • z - high-impedance value

Registers (reg) and nets can be declared to be scalars. This is the default, so if a range specification is not present in the declaration, then the item being declared is a scalar.
