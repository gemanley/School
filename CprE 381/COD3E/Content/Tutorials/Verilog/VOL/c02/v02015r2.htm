/* All text between the comment start "/*" token and comment end token - */
/* is treated as comment. This includes text of all types, including
newlines. However, it does not include the comment end token.

This is another way of saying that you cannot nest block comments. As is
demonstrated above, when we wanted to include the */ /* token in the
first line, we had to start the comment over with the next token.
We had to do that here, too.            */

/* Block comments are good for removing unwanted code without deleting it,
* as in the following example.
*/

     always @(ain or bin /* or cin */)
        newstate = func(ain, bin);
