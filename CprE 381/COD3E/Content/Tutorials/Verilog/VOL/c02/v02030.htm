Integer constants are specified with three components:

1. Size - an unsigned decimal number base
2. Base - a radix specifier: one of 'd, 'b, 'h, 'o
3. Value - an unsigned number, which consists of digits appropriate for the specified radix. The letters x and z are also allowed (in all radixes), as well as the _ character. Underscores may be used indiscriminately, and are ignored.
The size and radix are both optional, but if a size is present, the radix must be also. There may be spaces between each of the three constant components.
