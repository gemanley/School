The Verilog word for a multi-bit quantity is a vector. If arithmetic is performed on a vector, the vector value is considered to be an unsigned integer.

Both nets and registers can be declared as vectors. They are declared by using a range specifier in the declaration:

wire [msb:lsb] w1;
and
reg [msb:lsb] r1;
Both msb and lsb must be constant valued expressions. The bit order can be either big-endian or little-endian, i.e. both msb<lsb and msb>lsb are allowed.
