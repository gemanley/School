Verilog resolves the dangling else in the standard way. That is, if there is more than one if that the else could be associated with, it is associated with the lexically nearest one. 

    if (cond_1)
        if (cond_2)
            result = x;
        else
            result = y;
In the above, the else is associated with the inner if, as implied by the indentation.

If you want to associate the else with a different if, then you must use begin...end, like this:

    if (cond_1) begin
        if (cond_2)
            result = x;
    end
    else
        result = y;
