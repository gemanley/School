Event controls indicate that execution of the attached statement must wait until the specified event occurs.

@event_expression

Event expressions may be one of the following

event_identifier
net_or_register
posedge net_or_register
negedge net_or_register
event_expression or event_expression

Event controls have no analog in declarative code - they are only used in procedural code.

Examples:

 
	wire clk;
	reg state, newstate;
	event e1;
	...
	@e1			a = f(x);
	@state			out = g(state);
	@(posedge clk)		x = x + 1;
	@(negedge state)	;
	@(e1 or posedge clk or newstate)	state = newstate;
Note the use of parentheses in the above examples. It is necessary to use parentheses whenever the event expression contains any white space. So, to use posedge or negedge, you must use parentheses, as well as whenever the event expression consists of more than one event linked with "or".
