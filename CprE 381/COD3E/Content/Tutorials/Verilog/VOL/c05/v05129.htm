The repeat loop is a very simple construct. The loop body is executed as many times as indicated by the count_expr. The count expression is evaluated once, before loop execution begins, so the loop count is not affected by changes to any of the variables used in the expression.

If the repeat count is negative or x, the loop is not executed. Unlike the other loop statements, all repeat loops must eventually terminate.

Examples

repeat (5) s1;
repeat (f(x,y)) s2;
repeat (loopcount) s3;
