x <= #5 y + z;

This statement is executed as follows:

1. evaluate the right-hand side expression and save the result
2. put the assignment on a list of nba's to be performed after the specified delay (t+5 in this example)
3. at the designated time, finish executing all other events except for other non-blocking assignments (nba's)
4. perform the assignment
The next statement is executed at the current time (t).
