The simplest event expressions consist of an event identifier. An event identifier is an identifier which was declared as an event, as in the declaration above of e1.

event e1;
Declared events have no value, they just are impulses. The only way a declared event can occur is for an event trigger statement to be executed:

->e1;
When it occurs, any event control waiting on that event will be satisfied. Any new event control for that event which is subsequently executed will wait for the next execution of an event trigger for that event.

Declared events are rarely used.
