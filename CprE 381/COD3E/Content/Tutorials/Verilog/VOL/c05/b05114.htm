Here is another example: 

    reg [2:0] sel, mask;
    ...
    mask = whichbit ? 3'bx11 : 3'b11x;
    casex (sel & mask)
        3'b010: state1;
        3'b111: state2;
    endcase
This code would either compare sel[1:0] or sel[2:1] with 2'b00 and 2'b11, depending on the value of whichbit.

Question:
If whichbit == 0 and sel == 3'b011, which task will be executed?

The answer is state1. If whichbit == 0, then mask == 3’b11x and sel&mask == 3’b01x. Using casex, the x bits do not participate in the comparison, so 3’b01x will match 3’b010, causing state1 to be selected.
