In addition, the state machine has the following properties:

  • The 1-bit input is sampled on the rising clock edge.
  • State transitions are made after the clock edge.
  • The output is available after the clock edge.
  • The model is 0-delay.

                dout <= din;
                state <= `A;

        endcase
