The following timing check tasks are available. Click each to learn more about it. 



$nochange (reference_event, data_event, 
           start_edge_offset, end_edge_offset [,notifier]);
This sets up checks to determine if the data_event occurs during a given level of the reference signal. This timing check is different from the others, in that there is no interval which is checked. When the data_event occurs, if the reference signal has the given value, then an error occurs.

The reference event must be an edge, either posedge or negedge. If the data event occurs before the opposite edge on the reference signal occurs, then an error is indicated. The start_edge_offset and end_edge_offset arguments may be used to expand or shrink the prohibited time interval. Both offsets may be negative.

Example:

        $nochange (posedge clk, dataIn, 0, 0);
