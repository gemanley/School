The following timing check tasks are available. Click each to learn more about it.



$recovery (reference_event, data_event, limit [,notifier]);

This sets up checks for the minimum time between the reference event and the data event. The reference event must be edge-triggered, and starts the interval. The data event is any event and it ends the interval.

An error is reported if the following check is true:

        time(data_event) - time(reference_event) < limit
If the data_event and reference_event occur at the same time, the reference_event is updated before the check is made, usually resulting in a violation.

Example:

        $recovery (posedge clk, strobe, tmin);
