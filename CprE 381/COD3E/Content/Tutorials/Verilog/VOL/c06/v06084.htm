The data file must consist of the following:

1. binary or hexidecimal numbers

[@address] datavalue

where

address is a hexadecimal number (for both $readmemb and $readmemh)
datavalue is a binary or hexadecimal number which may contain x, z, and _, but does not contain a size or base

2. white space

3. comments (both types)

If an address appears before a datavalue, the datavalue will go in that memory location, and subsequent datavalues will go in subsequent memory locations, proceeding toward the last_index.

The $readmem task is a kludge, in the sense that it only works for memories, and the form of the input is very inflexible.
