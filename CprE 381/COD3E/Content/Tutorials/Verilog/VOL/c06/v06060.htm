The general format of an output task is: 

        $task_name(arg1, arg2, ...);

where argi is either

a format string or
an expression

Format strings and specifiers are used in all of the output tasks:
$display
$displayb
$displayh
$displayo		 $write
$writeb
$writeh
$writeo		 $monitor
$monitorb
$monitorh
$monitoro		 $strobe
$strobeb
$strobeh
$strobec

If an argument is a string constant, then it is interpreted as a format string, which is very much like a C format string. That is, it consists of literal characters and format specifiers, which will cause output conversion of following argument expressions.

Special characters can be printed by using escape sequences in the format string.
