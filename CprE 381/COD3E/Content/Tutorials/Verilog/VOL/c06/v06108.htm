There are a set of conversion functions for handling real values.

$rtoi(real_value)

Returns a signed integer, truncating the real value.

$itor(int_val)

Returns the integer converted to a real value.

$realtobits(real_value)

Returns a 64-bit vector with the bit representation of the real number. This follows the IEEE 754 64-bit representation for real numbers. $realtobits is used to pass real numbers across ports, since you cannot put a real number in a port list.

$bitstoreal(bit_value)

Returns a real value obtained by interpreting the bit_value argument as an IEEE 754 floating point number. This is the inverse of $realtobits.
