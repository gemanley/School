The following timing check tasks are available. Click each to learn more about it.



$skew (reference_event, data_event, limit [,notifier]);

This sets up checks for the offset of a signal from the reference event. The event which begins the interval is the reference event, and the event which ends the interval is the data event.

An error is reported if the following check is true

        time(data_event) - time(reference_event) > limit
If the data_event and reference_event occur at the same time, the reference_event is updated before the check is made, which will result in the timing check always less than or equal to the limit.

Example:

        $skew (clk, derivedClk, tdelta);
