The following timing check tasks are available. Click each to learn more about it.



$width (reference_event, limit [,threshold [,notifier]]);

This sets up checks for the width of a signal. The event which begins the interval is the given edge on the reference signal, and the event which ends the interval is the opposite edge.

An error is reported if the following check is true:

        threshold < time(opposite_event) - time(reference_event) < limit
The threshold argument is optional, but must be present if a notifier is present.

Examples:

        $width (posedge dataIn, tmin, tepsilon, notifyReg);
        $width (posedge reset, tresetmin, 0, notifyReg);
        $width (negedge set, tSet);
