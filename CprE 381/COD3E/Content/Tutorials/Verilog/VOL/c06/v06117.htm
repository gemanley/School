The specify block is the construct which is used to provide detailed timing information about a model. This is a separate section of a module which contains timing parameters and the specifications for how long paths within the module will take, the module path delays, and for timing constraints, the timing checks.

Specify blocks look like

    specify
        <specparam declarations>
        <module path declarations>
        <timing checks>
    endspecify

Specify blocks appear inside the module definition.
