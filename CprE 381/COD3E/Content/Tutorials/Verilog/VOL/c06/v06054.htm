Another example is the use of disable to implement a time out: 

    fork
        begin: op
            @ev do_some_op;
                disable timer;
        end
        
        begin: timer
            repeat (3) @(posedge clk) ;
            disable op;
        end
    join

In this example, the op block does the work, but it has to wait for some external event, ev. If the external event does not occur in 3 clock cycles, then the timeout stops the activity.
