This is an example of an AMD 2901 ALU. This comes from a translation from VHDL. The original model came from the '92 High Level Synthesis Benchmarks. The author was Sari Coumeri, a graduate student at CMU. There is a test module and the a2901 module. The test module instantiates the a2901 and creates a set of test vectors. It also checks the results, so the output of this test when it runs correctly is one line at the end which says that the test passed. Note that there is a $monitor statement which is commented out.
