The $time system function simply returns the current simulation time. Simulation time is a 64-bit unsigned quantity, and that is what $time is assumed to be when it is used in an expression.

Example:

        $display ("The current time is %d", $time);
        $display ($time," now the value of x is %h", x);

The value returned is scaled by the timescale units in effect for that module. See the discussion of time scale later in this chapter.

$stime (short time) is just like $time, except that it returns a 32-bit value of time. The low order 32 bits of the time are returned. If any high-order bits are non-zero, there is no warning or error message.
