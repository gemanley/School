A single module path statement may describe many module paths. This can be done by using the full connection operator, *>, and by using lists of source and destination specifiers. For example,

        (a, b, c *> x, y) = 10;
is equivalent to: 

        (a *> x) = 10;
        (a *> y) = 10;
        (b *> x) = 10;
        (b *> y) = 10;
        (c *> x) = 10;
        (c *> y) = 10;
