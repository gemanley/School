Rewrite module abc above so that the input is a 2-bit vector s[1:0], and the output is a 4-bit vector d[3:0]. Call this module vabc. Click Answer to see a solution. 

module vabc (d, s);
   input [1:0] s;
   output [3:0] d;

   not  (s1_, s[1]), (s0_, s[0]);

   and (d[3], s1_, s0_);
   and (d[2], s1_, s[0]);
   and (d[1], s[1], s0_);
   and (d[0], s[1], s[0]);
endmodule
