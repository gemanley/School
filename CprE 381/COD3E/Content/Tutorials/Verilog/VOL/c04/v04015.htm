There are two parts to port connections, the port definition and the port instance. Naturally enough, the port definition is found in the module definition port list, and the port instance is found in the module instantiation port list.

Port expressions are one of

  • identifier
  • bit-select
  • part-select
  • concatenation
  • constant
These are the same types of entities which can appear on the left-hand side of a continuous assignment statement.

A port expression can be specified in the port list as either the expression by itself, or with a port identifier using the .id(expr) notation. This is true in both the definition and the instance.
