OBJECTIVES

On completion of this chapter you will be able to:

  • predict the output of a gate level Verilog model given its inputs
  • describe how to correct a gate level Verilog model given its source code, inputs and output
  • write a Verilog gate-level model corresponding to a given simple schematic
  • determine the schematic for a gate level Verilog model given its source code
  • identify Switch-Level and User-Defined primitives in a Verilog model
