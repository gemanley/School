Given the monitor statement:
	$monitor($stime, ":  s2 = %h", s2);
for the following module:
	module zero3 (out, in, clk);
		output out;
		input in, clk;
		reg s1, s2, s3;

		assign out = s1==0 & s2==0 & s3==0;

		initial begin
			s1 = 0;
			s2 = 0;
			s3 = 0;
		end

		always @(posedge clk) begin
			s1 
				
				
				<= in; s2 <= s1;
			s3 <= s2;
		end
	endmodule
Where should the monitor statement be inserted? Click Done to check your answer.

a. before the statement s1 = 0;
b. after the statement s3 = 0;
c. before the statement s1 <= in;
d. after the assign statement
e. in its own initial block
f. either a or e
g. either b or c
h. none of the above
That's right. The monitor statement should be inserted either before "s1 = 0;" or in its own initial block.
