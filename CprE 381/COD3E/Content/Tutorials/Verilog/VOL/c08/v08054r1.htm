This system task is not part of the IEEE standard language but it is commonly implemented. It allows the model to check for the existence of a command line argument.

Command line arguments which are model-visible start with a "+". For example,

    VERILOG file1.v file2.v -v libfile +fullvis

The above command has a user-defined command line argument named fullvis. The model could find out if it was included on the command line by using $test$plusargs, as follows:

    if ($test$plusargs("fullvis"))      // if +fullvis
         $dumpvars;                     //     dump everything
    else                                // otherwise
        $monitor($time,,r1,,r2,,r3);   //     just monitor a few signals
