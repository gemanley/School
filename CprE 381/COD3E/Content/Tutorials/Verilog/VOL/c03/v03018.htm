Expressions are made up of operands and operators. Operands can be

  • constant
  • net
  • register
  • bit-select
  • part-select
  • concatenation
  • memory element
  • function call
Click any of these items to review the screen where it was introduced.

You can think of bit-select and part-select as unary operators, but they are not quite the same, since they can be used in places where general expressions are not allowed (like the left hand side of an assignment). The same is true for concatenations. So, Verilog generally treats them as separate classes of entities.
