An expression whose value is determined at compile time is a constant expression. Constant expressions are often used instead of constant numbers to document the meaning of the constant value. For example,

This... Is better than this...
reg [8*6:1] string6;

parameter buswidth = 24;
input [buswidth-1:0] bus;

 reg [48:1] string6;

input [23:0] bus;

Note that parameters can be used in constant expressions.
