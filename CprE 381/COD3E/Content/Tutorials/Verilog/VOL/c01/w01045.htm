What is wrong with the following module definition? 

	module A (p1, p2, , p4);
		input p1, p2;
		output p4;
		...
	endmodule
a. the missing port must be declared
b. missing ports are not allowed in module definitions
c. there must be at least one inout port
d. nothing is wrong

The correct answer is b. Missing ports are not allowed in module definitions.
