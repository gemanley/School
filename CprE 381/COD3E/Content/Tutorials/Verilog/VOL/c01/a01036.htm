Ports are Verilog structures that pass data between parent and child modules. Thus, ports can be thought of as wires connecting modules. The connections provided by ports can be either input, output, or bi-directional (inout).

Type Data Direction   
input parent->child
output child->parent
inout child<->parent
Ports can pass data from a parent to a child, or a child to a parent.
