Parameters are used wherever a constant expression is allowed. Common uses of parameters include vector range specifications, delay values, and case selector values.

Click the Example button to see how the following parameters might be used.

        parameter width = 8, msb = 7, lsb = 0;
        parameter delay = 100;
        parameter maxval = (1 << width) - 1;
        parameter r = 5.7;
        parameter state1 = 4'b0001, state2 = 4'b0010, 
                  state3 = 4'b0100, state4 = 4'b1000;
