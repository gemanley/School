Registers are storage elements. Values are stored in registers in procedural assignment statements. Registers can be used as the source for a primitive or module instance (i.e. registers can be connected to input ports), but they cannot be driven in the same way a net can.

Registers are declared in statements like this:

     reg [range] list_of_register_identifiers ;
or   integer list_of_register_identifiers ;
or   time list_of_register_identifiers ;
or   real list_of_real_identifiers ;
or   realtime list_of_real_identifiers ;

Typical register declarations would be:

     reg r1, r2;
     reg [31:0] bus32;
     integer i;
     real fx1, fx2;
