Are these two ways of redefining the parameter par2 equivalent?

a. Yes    No

      modB #(,5,) mb1 (p1, p2, p3, p4);

      and

      modB mb1 (p1, p2, p3, p4);
      defparam mb1.par2 = 5;

You can't do what this is attempting by means of a module instance parameter assignment. The defparam is the only way to change the value of only the second parameter in a module.
