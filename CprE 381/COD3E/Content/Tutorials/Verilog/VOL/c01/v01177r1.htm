Parameters are useful because they can be redefined on a module instance basis. That is, each different instance of module foo can have different parameter values. This is particularly useful for vector widths. For example, the following module implements a shifter:

     module shift (shiftOut, dataIn, shiftCount);
          parameter width = 4;
          output [width-1:0] shiftOut;
          input [width-1:0] dataIn;
          input [31:0] shiftCount;

          assign shiftOut = dataIn << shiftCount;
     endmodule

This is module can now be used for shifters of various sizes, simply by changing the width parameter. Parameters can be changed per instance in either of two ways.
