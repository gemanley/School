The defparam statement allows you to change a module instance parameter directly from another module. This is usually used as follows: 

    shift sh1 (shiftedVal, inVal, 7);   //instantiation of shift module
    defparam sh1.width = 16;        // parameter redefinition
Note that the defparam statement changes just the named parameter in the named module instance. The same parameter in all other instances of this module type remains unchanged.
