By definition, the top level module is not instantiated by any other module. For the top level module, the module name, which is really its type name, is also used as its instance name. Thus, in the following, module foo is the top level module.
    module foo;
      bar bee (port1, port2);
    endmodule

    module bar (port1, port2);
         ...
    endmodule
The child module is named bee, and it is of type bar.
