Writing user routines for use with Verilog models would not be interesting unless data could be passed back and forth between the model and the user routines. This is done by means of utility routines, which are part of the PLI.

Data can be passed as:

  • unsigned integers
  • reals
  • vectors up to 32 bits
  • vectors up to 64 bits
  • vectors over 64 bits
  • 4-value scalars or vectors
  • 120-value scalars
  • strings
