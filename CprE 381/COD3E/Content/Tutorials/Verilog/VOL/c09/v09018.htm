This example is a C routine which will print "Hello World from My Routine". 
/* ex_hello.c */
#include 
					
					
					
					
					
					
#include "acc_user.h"


int helloWorldFunction ()
{
        io_printf("Hello World from My Routine\n");
}
Note that io_printf is a utility routine which is part of the PLI. It acts just like printf, except that it puts its output both on stdout and the log file, if there is one.
