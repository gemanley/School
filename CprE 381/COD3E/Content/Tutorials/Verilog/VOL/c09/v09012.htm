When people talk about the PLI, they often mix several different things. The objects which are involved are:

  • Verilog source
  • Verilog simulator (or run-time library)
  • C program (or C++ or potentially some other language)
  • PLI header - veriuser.h
  • simulator-specific data structure
    • veriuser.c (Verilog-XL)
    • pli.tab (VCS)
  • simulator-provided utility routines
