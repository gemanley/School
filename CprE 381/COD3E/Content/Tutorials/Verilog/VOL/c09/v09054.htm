This example is a user-defined task which, when called, will set a given register to a given value. Both the register and the value are given as parameters to the system task.

The utility routines it uses in the C code are:

tf_getp(narg)
A function that returns the value of the nth parameter of the user-defined task. The value returned is 2-value (x and z values are returned as 0).

tf_putp(narg, new)
Change the nth parameter to the value given by new. If narg is 0, the function value is returned.

io_printf ( "In my_tf: old=%d, new=%d\n", old, new )
A print routine which prints to both stdout and the log file (if any).
