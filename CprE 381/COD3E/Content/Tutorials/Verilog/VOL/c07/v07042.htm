In a Moore machine, where the output is a function only of the state, the outputs could be produced by code that looked like this: 

    assign out = outfunc(state);
    function outfunc;
        input [1:0] state;
        case (state)
        `A:      outfunc = 1;
        `B:      outfunc = 0;
        `C:      outfunc = 1;
        default: outfunc = 1'bx;
        endcase
    endfunction

Alternatively, the output assignment could be done like this:

    assign out = state==`A ? 1 :
                 state==`B ? 0 :
                 state==`C ? 1 : 1'bx;
