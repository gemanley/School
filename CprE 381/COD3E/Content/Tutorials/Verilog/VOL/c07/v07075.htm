Many of the examples which we have used are delay-less, or zero-delay models. The advantage of zero-delay models is that they simulate faster than models which have a lot of delays in them.

The reason that zero-delay models are faster in execution can be seen by comparing the steps required to execute the following two functionally equivalent bits of code.

   ...					...
   assign #1 w1 = r1 + r2;		assign w1 = r1 + r2;
   assign #2 w2 = w1 + r1;		assign w2 = w1 + r1;
   assign #3 w3 = r1 - r2; 		assign w3 = r1 - r2;

   always@(strobe)			always@(strobe)
      r1 = #1 r1 + 3;			   r1 
					
					
					<= r1 + 3; ... ...
