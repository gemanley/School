It is common to make a single pipe stage module and use it repetitively, as follows: 

module pipeline (out, in, clock);
    output out;
    input  in, clock;
    wire   s1out, s2out;
    
    pipestage s1  (s1out, in,    clock),
              s2  (s2out, s1out, clock),
              s3  (out,   s2out, clock);
endmodule

module pipestage (out, in, clock);
    output out;
    input  in, clock;
    reg    out;
    
    always @(posedge clock)
        out 
					
					
					<= in; endmodule
Of course, this works just like the previous version, and the pipestage module is nothing more than a D flip-flop.
