Write a module which implements a 3-stage pipeline. Each stage is a single bit, and there is a single bit as input. The output should be 1 whenever all three stages contain a zero, which is the initial condition. The module template is: 
module zero3 (out, in, clk);
	output out;
	input in, clk;
	reg s1, s2, s3;

	assign out = s1==0 & s2==0 & s3==0;

	initial begin
		s1 = 0;
		s2 = 0;
		s3 = 0;
	end

	always @(posedge clk) begin
		s1 <= in; 
		s2 <= s1;
		s3 <= s2;
	end
endmodule
