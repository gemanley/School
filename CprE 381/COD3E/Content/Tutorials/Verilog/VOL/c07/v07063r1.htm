An interesting variation is the following: 

module pipeline (out, in, clock);
    output out;
    input  in;
    reg pipe[1:2];
    
    always @(posedge clock)
        {out, pipe[2], pipe[1]} = #d {pipe[2], pipe[1], in};
endmodule
Use of the intra-assignment delay is possible here because there is only one statement in the always block.
