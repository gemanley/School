What is the value of z printed by this module?

	module t;
	reg x, y, z;
	initial  begin 
		x = 1; 
		y = 0; 
	#1 	$display(z);
	end
	always @(posedge x)
		z = ~y;

	endmodule
1. 0
2. 1
3. x
4. z
5. either 1 or x
6. either 1 or z
7. either 0 or x
8. either 0 or z
The correct answer is item 5. There is a race between the initial block and the always. If the initial block is executed first, the test for posedge x will wait for the next rising edge, not the one that happened at time 0.
