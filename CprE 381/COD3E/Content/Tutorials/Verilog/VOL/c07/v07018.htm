Clocks written with procedural code are by far the most common. Probably the simplest loop is this: 

    always begin
        #period/2 clk = 0;
        #period/2 clk = 1;
    end
Here are some variations: 
    initial clk = 0;
    always #period/2
        clk = ~clk;

    initial begin
        clk = 0;
        forever #period/2
            clk = ~clk;
    end
