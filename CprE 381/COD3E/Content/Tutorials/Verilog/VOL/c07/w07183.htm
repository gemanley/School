Which of the statements about the following code fragment are true? Assume that "z" changes at "negedge clk" and "clk" has a period of 20 time units.

	always @(posedge clk) begin
		x 
				
				
				<= f(y); @(y) ; end always @(posedge clk) begin y <= g(z);
		@(z) ;
	end
1. This code will work.
2. This code is efficient. It reduces the simulation activity if "y" or "z" does not change on every "clk."

Both statements are true. Statement 1 is true because the non-blocking assign causes the "@(y)" and "@(z)" to be reached before the values of "x" or "y" change. Statement 2 is true because the "@(y)" and "@(z)" cause the "always" block to suspend until "y" or "z" change.
