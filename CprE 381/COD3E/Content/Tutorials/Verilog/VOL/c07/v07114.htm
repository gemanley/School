There is no substitute for measuring the performance of a model and experimenting with alternatives. However, besides buying a faster simulator, there are a number of things which can be done to improve the execution time of a simulation model. Here are some techniques to improve efficiency that we will discuss on the following screens.

  • Compile Time vs. Run Time
  • Reduce the Number of Events
  • Use Higher Levels of Abstraction
  • Don't Split Vectors
  • Avoid Bi-Direct Primitives
  • Minimize RHS Evaluations in Continuous Assigns
  • Optimize Always @(event_expression) Blocks
