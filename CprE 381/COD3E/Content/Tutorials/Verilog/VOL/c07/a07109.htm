Which of the following is an example of a race? Click Done to check your answer.

a.
module t;
	...
	assign x = f(y);
	assign z = f(x);
	...
    endmodule
b.
module t;
	...
	always @(posedge clk)
		x = f(y);
	always @(posedge clk)
		z = f(x);
	...
    endmodule
1. a
2. b
3. a and b
