Link mode

Link an already bound design unit.

$ ghdl --link [options] primary_unit [secondary_unit]

This performs only the second stage of the elaboration mode: the executable is created by linking the files of the object files list. This mode is available only for completness. The elaboration mode is equivalent to the bind mode followed by the link mode.
