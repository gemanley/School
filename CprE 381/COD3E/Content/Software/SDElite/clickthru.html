Install MIPS SDE-lite

Before installing this software, you must have agreed to the terms and conditions of the license agreement.

  • For users inside the United States
  • For users outside the United States

WINDOWS

  • NOTE: You must install Cygwin before installing or using SDE-lite.
    See http://cygwin.com for instructions and the files to do this. Alternatively you can use the Network Installer included on this CD.
    The MIPS SDE Programmer's Guide contains more information.
  • Click here to open a tar.gz file containing SDE-lite for Windows
    To use the software, create a new folder without any spaces in the name and extract the contents of this archive into the folder. Then execute the script "bin/sdesetup.sh" using the Cygwin shell.

LINUX

  • Click here to open a tar.gz file containing SDE-lite for i386/Linux systems.
    To use the software, extract the contents of this archive into a good location (e.g. "/usr/local/sdelite" or "/home/myname/sdelite" and then execute the script "bin/sdesetup.sh".
