Com S 363: Introduction to Database Systems
Introduction to Database Systems

 

1. Project goal 

This project is meant to be a simple one. The purpose of this project is three fold.

  • Make sure that you have installed and configured all the software artifacts to be used in the course. For this, visit the [Software Systems] web page.
  • Learn how to submit projects. For this please visit the [Projects] web page. The methodology for submission of all projects in the course will be the same.

2. Project details 

Create a folder P0 anywhere on your system. This folder will contain all the files to be submitted for Project P0. Do not use this folder to store any other files. (In future you will create folders P1, P2, P3, and P4 for the same purpose for other projects.)

Visit the [Software Systems] web page and complete all tasks detailed there. We are very hopeful that things will go right. Assuming this, the task is not as big as it seems - so don't feel intimidated. When you are done you will have the following 5 files in folder P0:

  • ComS363_XQuery(Saxon)_Demo.htm
  • Java_Demo_Output.txt
  • ComS363_MySQL_Demo_Output.htm
  • JDBC_Demo_Output.txt
  • ComS363_Test_ID_Function_in_XQuery(Saxon)_Output.htm

The name of these files have been highlighted on the [Software Systems] web page.

3. Submit the project

Follow the procedure outlined on the [Projects] web page. Note that the submission is in two steps.

Step 1. The electronic submission. This part of submission authenticates the date of your submission. To complete this step, submit the contents of folder P0.

Step 2. The hardcopy submission. Follow this by a hardcopy submission. For this project the hardcopy will simply be a coversheet that you can download from [Projects] web page. Fill the coversheet and on the back list the names of all files you are submitting.

We assume that you will make reasonable effort to submit the hardcopy ASAP after your electronic submission. Normally you would submit the hardcopy in class. But when need arises, a drop box for your use is available outside Room 105 Atanasoff. Note that access to the first floor of Atanasoff Hall is available during usual office hours.
